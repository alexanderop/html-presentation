HTML Presentations

Zero-dependency slides from a single prompt

The Problem

  • Slide tools require accounts, installs, or subscriptions
  • Markdown-to-slides tools need build steps and config
  • Sharing means exporting PDFs or sending links to hosted services
  • You just want to present something quickly

The Solution

A Claude Code skill that generates a single, self-contained HTML file you can open in any browser.

  • No dependencies
  • No build step
  • No slides app needed
  • Works offline

How It Works

  1. You say create a presentation about X
  2. Claude generates markdown with slide separators
  3. Each slide becomes an HTML <div> block
  4. Everything is embedded in a single .html file
  5. It opens in your browser automatically

Features

Navigation

  • Arrow keys & space
  • Home / End keys
  • 1-9 to jump to slide
  • Touch & swipe support

Design

  • Dark theme
  • Gradient headings
  • Progress bar
  • Fullscreen mode

Rich Content Support

Presentations can include all standard markdown elements:

  • Code blocks with syntax highlighting styles
  • Tables for structured data
  • Blockquotes for callouts
  • Two-column layouts for comparisons
  • Images and links

Code Blocks

Styled and readable, right in the slides:

// Install the skill
npx skills add alexanderop/html-presentation

// Then just ask Claude
"Create a presentation about microservices"
"Make slides about our Q4 roadmap"
"Present about TypeScript best practices"
